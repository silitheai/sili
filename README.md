## 🚀 Deployment

### One-Line Install
```bash
curl -sSL https://raw.githubusercontent.com/silitheai/sili/main/install.sh | bash
```

### Manual Nexus Setup
1. Clone the repository.
2. Initialize: `python3 -m venv venv && source venv/bin/activate`
3. Link Dependencies: `pip install -r requirements.txt`
4. Awake Browsers: `playwright install chromium`
5. Configure Soul: `python3 setup.py`
6. Activate Sili: `python3 telegram_bot.py`
